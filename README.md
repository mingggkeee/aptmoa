# Aptmoa
공공데이터와 카카오맵 API를 활용한 아파트 실거래가 정보 및 리뷰 제공 사이트

### 배경
힘들어진 내 집 마련 - 급격하게 오른 집값으로 이전보다 더욱 힘들어진 내 집 마련 힘들게 사는만큼 신중하게 좋은집을 구하고 싶다

### 목적
좋은 집의 조건을 고려하여 정보를 제공한다
- 집 주변에 편의시설이 있는지 보여주는 인프라지수
- 실제 거주 경험이 있는 사람의 신뢰도 높은 리뷰
